1. How to run the program?
--------------------------------------
Simply run the "PA-1-Source.exe" file. 

However, the program is designed to append to the Conclusions.txt file instead of overwriting. It is recommended that you delete 'Conclusions.txt' before running.

--------------------------------------
The spreadsheets and text file are all generated by the program.
